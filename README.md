# Repositório de aulas do Streamlit

Este repositório contém uma coleção de aulas e exemplos práticos utilizando o Streamlit, uma biblioteca de código aberto em Python para a criação de aplicativos web interativos de forma rápida e fácil.

#

## Aula 01 — Introdução e Estrutura Básica do Streamlit

Nesta aula foi iniciada a série prática de 12 dias focada em **Streamlit**, com o objetivo de consolidar os fundamentos necessários para a criação de aplicações de dados em Python.

### Conteúdos abordados
- Importação e estrutura inicial de uma aplicação Streamlit
- Criação de títulos, cabeçalhos e subcabeçalhos
- Exibição de textos dinâmicos com `st.write()`
- Uso de Markdown para formatação de conteúdo
- Diferença entre texto formatado (`st.markdown`) e texto puro (`st.text`)

### Objetivo da aula
Estabelecer uma base sólida para o desenvolvimento de dashboards interativos e aplicações analíticas, garantindo organização visual, clareza da informação e boa experiência do usuário.

### Exercício — Aula 01 | Página de Perfil em Streamlit

Como exercício prático da Aula 01, foi desenvolvida uma aplicação simples em Streamlit simulando uma **página de perfil pessoal**.

#### Objetivos do exercício
- Fixar o uso de títulos, cabeçalhos e subcabeçalhos
- Aplicar formatação de texto com Markdown
- Organizar informações de forma clara e visualmente agradável
- Introduzir noções iniciais de experiência do usuário (UX)

Mesmo sendo um exercício introdutório, ele reforça conceitos fundamentais que serão reutilizados ao longo da série na construção de dashboards e aplicações de dados mais avançadas.
